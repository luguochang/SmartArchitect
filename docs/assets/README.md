## Tips for Creating Good Demo GIFs

### Recommended Tools
- **Windows**: ScreenToGif (free, open source)
- **Mac**: Kap (free, open source) or LICEcap
- **Linux**: Peek (free, open source) or SimpleScreenRecorder + ffmpeg

### Recording Guidelines
1. **Resolution**: Use 1920x1080 or 1280x720 for recording
2. **Frame rate**: 15-30 FPS is sufficient for demos
3. **Duration**: Keep it under 20 seconds for file size
4. **File size**: Aim for under 5MB per GIF (use optimization tools)
5. **Quality**: Balance between quality and file size

### GIF Optimization Tools
- [ezgif.com](https://ezgif.com/optimize) - Online GIF optimizer
- [gifsicle](https://github.com/kohler/gifsicle) - Command-line tool
- ImageOptim (Mac) - Drag and drop optimization

## Screenshot Guidelines

### Taking Screenshots
1. Use a clean browser window (no bookmarks bar, extensions)
2. Use a professional theme (Light or Geist recommended)
3. Include some sample content (not empty canvas)
4. Capture at a reasonable resolution (at least 1280x720)

### Screenshot Content Suggestions

**Canvas View:**
- Show 5-7 connected nodes
- Display the code editor on the side

- Use a light theme for clarity

**Chat Generator:**
